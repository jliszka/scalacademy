Scalacademy

A programming tutorial for absolute beginners.

Easy Installation

  1. Download Scalacademy by clicking here.
  2. Click on the downloaded file to install it.

Using Scalacademy

  1. Open the Finder. You should see a folder called scalacademy in Downloads.
  2. Inside that folder is a file called scalacademy.command. Double-click it.
  3. A window will open and some stuff will happen. It will take a few the minutes the first time you run it while it downloads Scala for you.
  4. When stuff is done happening, you will see a scala> prompt. Type start and hit enter.
  5. Follow the prompts on the screen.

Tips

  1. To start at any level, for example level 3, type level(3) and hit enter.
  2. To restart the current level, type restart and hit enter.
  3. To repeat a prompt, type repeat and hit enter.
  4. To quit, type :quit and hit enter (mind the colon!).

Uninstall

Just delete the folder called scalacademy.
